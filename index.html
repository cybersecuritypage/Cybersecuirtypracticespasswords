Understanding MFA and Password Managers

Strong account security goes beyond using a password. Tools like Multi Factor Authentication (MFA) and password managers add extra layers of protection against unauthorized access.

What Is Multi‑Factor Authentication (MFA)?

Multi Factor Authentication (MFA) requires more than one way to verify your identity when signing in. Even if a password is stolen, MFA helps prevent attackers from accessing the account by requesting:

  • Something you know: a password or PIN
  • Something you have: a phone, security key, or authenticator app (such as Microsoft Authenticator or Google Authenticator)
  • Something you are: biometrics such as fingerprint or face recognition

What Is a Password Manager?

A password manager keeps all passwords safe in one encrypted vault, which can only be unlocked with one strong password.

Common Types of Password Managers

  • Dedicated password manager apps

    Tools designed to securely store and generate passwords across devices using encryption and often MFA. Examples: 1Password, LastPass

  • Browser‑based password managers

    Built into web browsers such as Chrome, Edge, Firefox, and Safari/iCloud Keychain. Their security depends on how well the browser and device are protected, especially on shared devices.

  • Device or operating‑system password managers

    Tools built into phones and operating systems (such as iOS and Android) that sync passwords across linked devices.

Important Note for Work Accounts

While MFA and password managers are commonly used for personal accounts, workplace accounts must always follow organizational policies and use approved tools.

PASSWORD BEST PRACTICES

How to Create a Strong Password

Strong passwords help protect your accounts from unauthorized access. One of the most effective ways to create, and remember, a secure password is to use a passphrase instead of a single word.

Tip: Use a Passphrase

Combine three to four random words with numbers and symbols to create a passphrase that is both strong and easier to remember.

Make each passphrase unique by adding a small twist for each account. An example of a twist is a letter which represents the type of account or service. Avoid using exact site or service names.

Example:

  • Email: SilverCarRunsFast!E5
  • Cloud: SilverCarRunsFast!C5
